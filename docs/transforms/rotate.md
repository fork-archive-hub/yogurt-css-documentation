> Transforms

## Rotate

Utilities for rotating elements with transform.

### Class

| class |  | css properties |
|:--|:--|:--|
| rotate-0 |  | --transform-rotate: 0 |
| rotate-45 |  | --transform-rotate: 45deg |
| rotate-90 |  | --transform-rotate: 90deg |
| rotate-180 |  | --transform-rotate: 180deg |
| -rotate-180 |  | --transform-rotate: -180deg |
| -rotate-90 |  | --transform-rotate: -90deg |
| -rotate-45 |  | --transform-rotate: -45deg |
