## Z-Index

Utility for sets the z order (stack order) of a positioned element.

### Class

| class |  | css properties |
|:--|:--|:--|
| z-0 |  | z-index: 0 |
| z-10 |  | z-index: 10 |
| z-20 |  | z-index: 20 |
| z-30 |  | z-index: 30 |
| z-40 |  | z-index: 40 |
| z-50 |  | z-index: 50 |
| z-auto |  | z-index: auto |
