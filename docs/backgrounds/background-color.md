> Backgrounds

## Background Color

Utilities for manipulating an element's background color.

### Usage

| class |   | css properties |
|:--|:--|:--|
| bg-transparent |  | color: transparent |
| bg-black |  | color: #000 |
| bg-white |  | color: #fff |
| bg-`{color}`-`{modifier}` |

> See [Colors](../customization/colors.md) for `{color}`-`{modifier}`.
